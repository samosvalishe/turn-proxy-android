## 3.0.0

### BREAKING CHANGE
> **При обновлении до 3.0.0 настройки сбрасываются.** Переезд с модели "профили" на "серверы" сменил формат хранения (ключи `AppPreferences`, снимок серверов как единственный источник конфига). Старые данные не мигрируют - серверы, SSH-доступы и тоггл-настройки нужно завести заново.

Мажорный релиз. Переход с модели "профили" на "серверы", новые фичи (шаринг по ссылке, бэкап настроек, мастер установки), большой рефакторинг архитектуры по слоям и упрочнение прокси/SSH.

### Features

**Шаринг и импорт**
- Выдача доступа по ссылке/QR и импорт - `freeturn://`-ссылки, генерация QR, импорт без SSH (`53d624f`, `a49c591`)
- Постоянный Client ID устройства в аргументах ядра + авторизация по Client ID на сервере (`417349b`, `a49c591`)
- Online-статус пира гаснет по времени, не только при перезагрузке (`1631161`)

**Бэкап настроек**
- Экспорт/импорт настроек в зашифрованный паролем файл (`b192f9b`)
- Персист приватного режима + восстановление тоггл-настроек (`383229f`)

**Серверы**
- Мастер установки self-hosted сервера (`8a56fb3`)
- Клонирование сервера из меню хаба (`0716dba`)
- Вкладка "+" - экран добавления сервера (`85a3648`)
- Apply-модель: единый SSH-лог и apply для неактивных серверов (`a7623c8`)

**UI / навигация**
- Expressive-герой на главном - морф формы, glow, живой статус (`0b6bef5`)
- Empty-state без серверов: прячем лист и тоггл (`9b869be`)
- Per-tab back-stacks с Material expressive motion (`6c19e13`)
- Компактный нижний бар без подписей (`abfa2f5`)
- Settings: разделы "Приложение"/"О проекте", экран "Продвинутые", отладочная инфа, ручной DNS, nerd-режим (`a5c0536`, `e2f9b0c`, `376ee26`)
- Режим отладки включён по умолчанию (`2db3984`)

**Ядро / прокси**
- Единый билдер argv `CoreArgs.client(...)` - UI-команда совпадает с запускаемой (`376ee26`)
- Маскировка секретов в argv для лога (`redactForLog`) (`ef437e0`)
- Ручная вставка WG-конфига в режиме VPN (`8e9d092`)

### Fixes

**Прокси / надёжность**
- Single-flight запуска ядра, убийство зомби при стопе в окне старта (`b3b826f`)
- Сериализация подъёма/остановки WG + откат туннеля при провале (`393f741`)
- Wakelock без таймаута - сессия может длиться дольше суток (`d668645`)

**SSH**
- Сериализация живой SSH-сессии mutex'ом (`ede1a4c`)
- Не рестартить удалённый сервер при смене активного (`dfc46c9`)
- Сброс стека "Настроек" при смене активного сервера (`9493c4c`)

**Безопасность обновлений**
- Проверка подписи APK перед установкой (`f84095a`)
- `latestApkUrl` как `@Volatile` (`0aecc07`)
- Закрыт `ProxyReceiver`, чистка proguard по аудиту (`8732be9`)

**Setup / прочее**
- Валидация клиентского WG-конфига при повторной установке (`185b509`)
- Не пускать тапы в контент во время перехода (`8a6aefa`)
- Не вибрировать при выключенном системном виброотклике (`7afdc97`)
