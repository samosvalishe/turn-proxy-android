# Changelog

## v1.3.0

### Что нового

**Приватный режим**
При включении скрывает IP-адрес сервера, VK-ссылку и порты на экране настроек — поля заменяются звёздочками. Удобно для скриншотов и демонстрации экрана без раскрытия конфигурации.

**Кастомное ядро — улучшенный интерфейс**
Карточка загрузки собственного ядра полностью переработана: теперь файл проверяется на корректность ELF-формата перед использованием. Если файл не подходит — сразу видна понятная ошибка прямо в карточке, без скрытых сбоев.

**Обновлённое ядро**
Встроенное ядро обновлено (сборка от cacggghp). Размер уменьшился с 9.5 МБ до 8.8 МБ.

### Улучшения интерфейса

- Переключатель TCP/UDP заменён на `SegmentedButton` — стало нагляднее и удобнее
- На главном экране в карточке настроек теперь отображается текущий транспорт (TCP/UDP)
- Поле VK-ссылки стало однострочным — без лишнего пространства
- Поле порта принимает только цифры
- Терминология SSH: «подключено» → «сопряжено» — точнее отражает состояние соединения

### Исправления

- Слайдер потоков больше не обрезает значения из-за ошибки округления (`roundToInt` вместо `toInt`)
- Исправлено условие показа онбординг-экрана при первом запуске
- Устранена лишняя вибрация на splash-экране
- Устранена утечка coroutine scope в ProxyService

### Безопасность

- Закрыта возможность shell-инъекции через параметры `listen`/`connect` при управлении сервером
