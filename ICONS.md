# Список иконок для миграции с material-icons-extended на XML (Material Symbols)

Источник: [Google Fonts Icons](https://fonts.google.com/icons) → вкладка Android → скачать XML в `res/drawable/`.

| Иконка | Стиль | Назначение |
|---|---|---|
| Home | Filled | Главная — selected |
| Home | Outlined | Главная — unselected |
| Storage | Filled | Сервер — selected |
| Storage | Outlined | Сервер — unselected |

| PhoneAndroid | Outlined | Клиент — unselected |
| Security | Filled | Onboarding, навигация |
